# Youth Portal Monorepo (TS)

A TS monorepo with React (Vite), Node.js (Express), and Firebase Authentication.

## Tech Stack
- **Client**: React 18, Vite, Tailwind CSS, Ant Design, React Query, Axios, React Router, TypeScript
- **API**: Node.js, Express, CORS, dotenv, TypeScript, ts-node
- **Auth**: Firebase Authentication (Email/Password + Google Sign-In)
- **Monorepo**: Single `package.json` at root, Yarn, Concurrently, Nodemon

## Setup

1. Install dependencies:
   ```bash
   yarn install
   ```

2. Create a `.env` file:
   ```bash
   PORT=5000
   ```
